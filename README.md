# 🌍 SkillPilot AI

### 🤖 AI-powered Career Decision & Roadmap System

SkillPilot AI is an AI-driven web app that helps users choose the right career path by generating personalized roadmaps, analyzing career scope, comparing skills, and giving smart AI-based recommendations.

---

## 🚀 Features

### 🎯 Core Features
- AI-generated learning roadmaps (Beginner → Advanced)
- Career scope analysis (Demand, Salary, Future growth)
- Skill comparison tool (AI vs Web Dev etc.)
- Personalized AI mentor chatbot
- Career recommendation system

---

### 🧠 AI Features
- Smart decision-making engine
- Career scoring system (0–100)
- Personalized suggestions based on interests & strengths
- Urdu + English language support

---

### 🧪 Extra Features
- Skill test for career recommendation
- YouTube learning resources
- AI mentor for career guidance

---

## 🛠 Tech Stack

- Python 🐍
- Streamlit ⚡
- Groq API 🤖 (LLM AI)
- OpenAI-compatible API client

---

## 🎯 Problem Statement

Students and beginners often struggle with:
- Choosing the right career path
- Lack of structured learning roadmaps
- No personalized career guidance
- Overwhelming online resources

👉 SkillPilot AI solves this by providing **AI-powered personalized career guidance**.

---

## 🧪 How It Works

1. User enters a skill (e.g., AI, Web Dev)
2. AI generates roadmap + career analysis
3. System evaluates user interests & strengths
4. AI suggests best career path
5. Optional skill test improves recommendations
